# AI-Powered Cold Email Generator

## Description

This **AI-Powered Cold Email Generator** automates the creation of personalized cold emails for job applications. It integrates **LangChain** for natural language processing, **ChromaDB** for querying portfolios, and **Streamlit** for building an interactive user interface. The tool dynamically matches job descriptions with a user's portfolio to generate tailored cold emails, allowing users to effectively pitch themselves to potential employers.

---

## Features

- **Personalized Cold Emails**: Generates job-specific cold emails based on your portfolio and job description.
- **Portfolio Integration**: Matches your portfolio projects based on the job description.
- **ChromaDB for Portfolio Management**: Efficient querying and management of your portfolio data.
- **Streamlit Web Interface**: A user-friendly interface for seamless interaction.
- **Customizable**: Easily extend and adapt the system to your needs.

---

## Technologies Used

- **LangChain**: For natural language generation and handling prompt chains.
- **ChromaDB**: A vector database for storing and querying portfolio data based on similarity.
- **Streamlit**: Framework for building interactive web applications in Python.
- **Python 3.x**: Programming language used for development.
- **Hugging Face Transformers**: For generating embeddings and processing the text.
